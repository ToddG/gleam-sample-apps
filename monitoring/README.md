# instructions

## deps

* [mise](https://mise.jdx.dev/)
* gleam

## quickstart

```bash
mise run app-build
mise run containers-start
mise run containers-stop
```

## tasks

```bash
cd gleam-sample-apps/monitoring

14:58 $ mise tasks
Name              Description
app-build         build the instrumented app
containers-start  run all servers
containers-stop   stop all servers
```
